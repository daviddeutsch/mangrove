What data is stored?

In order to identify and correctly service whoever is calling, connecting to a source - no matter in which way - requires transmitting the following data:

  • a UUID (a unique string of characters) generated from the site address as an identifier
  • PHP version
  • Database type and version
  • CMS version (if applicable)

No further details are stored.

Additional choices (such as activating automatic updates), however, may require further data to be stored.

We value your privacy and security and will always inform you about the data we store ahead of time.
